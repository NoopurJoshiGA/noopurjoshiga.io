My Work

'Boke' is full-stack application developed which serves as a networking platform, allowing users to register as either a photographer or a model, search and filter users based on their location, upload and comment on images.

I used Trello to formulate a plan and note down user stories and Sketch to create wireframes. I used a mobile-first approach to develop the application, and later ensured that it was fully responsive on tablet and desktop. Additionally, I used three external API’s - Postcodes.io, FileStack and Leaflet.

This project was built as my final project for General Assembly's Web Immersive Programme.
