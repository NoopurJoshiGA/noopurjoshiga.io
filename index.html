My Work

'Boke' is full-stack application developed which serves as a networking platform, allowing users to register as either a photographer or a model, search and filter users based on their location, upload and comment on images.

I used Trello to formulate a plan and note down user stories and Sketch to create wireframes. I used a mobile-first approach to develop the application, and later ensured that it was fully responsive on tablet and desktop. Additionally, I used three external API’s - Postcodes.io, FileStack and Leaflet.

This project was built as my final project for General Assembly's Web Immersive Programme.

I worked in a pair to create a MEAN full stack application which automates a users travel itinerary based on the location, duration, type of accommodation and user selected activities. I worked alongside my teammate to construct a project timeline, write thoughtful user stories on Trello, and closely collaborate to manage the GitHub branches, conflicts and merges in our code.

The application is created using a Mongo database, NodeJS, ExpressJS and AngularJS. The project is styled with Bulma CSS framework, includes authentication and automated tests built with Mocha and Chai.

This game was built as my third project for General Assembly's Web Immersive Programme.
